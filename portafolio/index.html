Presentación del Portafolio

Este portafolio digital tiene como propósito funcionar como el contenedor de las evidencias académicas del curso CNO V: Seguridad Informática.

El curso se enfoca en fundamentos de ciberseguridad, análisis de vulnerabilidades y pruebas de penetración, fortaleciendo una formación técnica orientada a la protección de la información.

El portafolio permite documentar de forma estructurada el proceso de aprendizaje a lo largo del periodo académico primavera 2026.

Perfil del Estudiante

Estudiante de la carrera de ingeniería en tecnologías de la información (ITI) de la Universidad Politécnica de San Luis Potosí (UPSLP), con interés en seguridad informática, ciberseguridad, investigación técnica y análisis de riesgos en sistemas.

Información Técnica

Este portafolio fue desarrollado como una aplicación web con fines de documentación académica, aplicando principios básicos de diseño web y presentación profesional.

  • Tecnologías utilizadas: HTML5 para la estructura semántica, CSS para el diseño visual y JavaScript para la interacción del usuario.
  • Plataforma de publicación: GitHub, utilizando alojamiento estático con disponibilidad pública mediante protocolo HTTPS.
  • Control de versiones: Git para la gestión de cambios y mantenimiento del historial de desarrollo del portafolio.
  • Seguridad básica implementada: Uso de HTTPS, validación de formularios del lado del cliente y separación de archivos (HTML, CSS y JS) para una mejor organización del código.
  • Enfoque del desarrollo: Documentación académica, usabilidad, accesibilidad básica y diseño visual.
